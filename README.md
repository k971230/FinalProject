# FinalProject
Spring MVC 모듈을 사용하여 그룹웨어 사이트를 제작한 팀프로젝트입니다.

# 개발 환경

OS: Windows 10
Language: Java 1.8.0
DBMS: Oracle 18c
Framework: Spring 5.3.22, Mybatis, Tiles, Bootstrap
Server: Tomcat 9.0
Front-End: HTML, CSS, javascript, jQuery
